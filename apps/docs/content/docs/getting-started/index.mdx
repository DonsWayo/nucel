Get up and running with Nucel in minutes. This guide will walk you through everything you need to know.

## Overview

Nucel simplifies deploying modern web applications to AWS using serverless architecture. With support for Next.js, SvelteKit, and React Router, you can deploy production-ready applications without infrastructure complexity.

## Why Nucel?

- **Zero Configuration** - Automatic framework detection and optimal AWS setup
- **Multi-Framework** - Native support for Next.js, SvelteKit, and React Router
- **Cost Efficient** - Pay only for what you use with serverless architecture
- **Production Ready** - Auto-scaling, global CDN, and enterprise-grade security
- **Full Control** - Your AWS account, your infrastructure, your data
- **Open Source** - Fully transparent, no vendor lock-in
