## Welcome to Pulumi AWS Frameworks

This documentation covers a collection of Pulumi components designed to simplify deploying modern web applications to AWS. Our packages support:

- **Next.js** - Full-stack React framework with OpenNext
- **SvelteKit** - Fast and modern Svelte framework  
- **React Router v7** - The latest React Router with server-side rendering

## Prerequisites

Before you begin, ensure you have:

1. **Node.js 18+** installed
2. **Pulumi CLI** installed and configured
3. **AWS Account** with appropriate permissions
4. **AWS CLI** configured with credentials

## Core Concepts

All packages share common patterns:

- **Serverless Architecture** - Lambda functions for SSR
- **CDN Distribution** - CloudFront for global content delivery
- **Static Asset Hosting** - S3 for efficient static file serving
- **Infrastructure as Code** - Version-controlled, reproducible deployments
